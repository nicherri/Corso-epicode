Inserire codice js inline

Posso inseire delle istruzione javascript direttamente in un elemento html, ad esempio asssegnando a un attributo che appresenta un eveteo (qualcosa che succede)

Altro approccio per inserire il codice inline, ma è valido solo per pr i link

Cliccami

Inserisco blocchi interi di codice js

Pro: nel caso in cui abbimao poco codice javascript lo metto subito in relazione con i tag html con cui devo interagire

Contro: nel caso- comune- in cui il codice js sia minimamento complesso ( cariabili, funzioni, oggetti0 non risulta comodo)

Tag head o tag body? il primo ordine di considerazioni: performance
secondo ordine di considerazione -> che il tag html su cui js sia già stato parsato ( letto e quindi diponibile) -> questo spiega perchè gli scrip sono in fondo alla pagina prima della chiusa di body.

Console

La console è uno strumento fondamentale per scrivere codice javascript ( in qualsiasi linguaggio di programmazione)

La condole ci permette di fare il debug cioè di controllare il flusso del nostro codice

spesso la console si usa per tenere traccia dell'evoluzione del programma o per verificare valori e trovare errori
