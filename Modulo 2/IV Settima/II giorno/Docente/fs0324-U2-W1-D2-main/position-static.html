POSITION: STATIC

Il valore "static" è il default della proprietà "position". Un elemento dotato di position: static NON è posizionato nella pagina in alcun modo particolare, ma seguirà sempre il normale "flow" della pagina. Si dichiara esplicitamente in pochi casi, magari quando ci sia bisogno che un elemento figlio non erediti la proprietà oppure nel caso sia necessario differenziarne il valore dentro una media query.

Questo div è posizionato con position: static
