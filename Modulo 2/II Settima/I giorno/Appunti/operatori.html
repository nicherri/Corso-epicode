Operatori

Tramite gli operatori posso combinare valori tra loro

Gli operatori deteminano il risultato di un' operazione

Classificazionedegli operatori in base al numero di valori che possono combinare: unari, binari, ternario

operatori aritmetici

Operatori binari perchè combinano due valori (numeri)
+ addizione
- sottrazione
/ divisione
* moltiplicazione
% modulo o resto

operatori unari

Operatori numerici possono essere unari, cioè combinano solamente un valore
++ incremento
-- decremento
-negazione -> x= 5 -> -x=-5
se ci sono due valore - è sottrazione, se c'è solo un valore è negazione

-> incremento e degremento le posso apllicare solo alle varibili
-> è molto importante la posizione dell'elemento (se a sinistra oppure e destra dell' operando)
-> se l'operazione è a sinistra dell' operando viene eseguita prima di valutare l'espressione
-> se l'operazione è a destra dell' operando l'operazione viene eseguita dopo

operatori logici

Consento di combinare eprssione booleane [true| false]
&& -> operatore binazio -> and
|| -> operatore binario -> o
! -> operatore unario -> not
se ci sono due valore - è sottrazione, se c'è solo un valore è negazione

operatori ternario

Pemette di combinare tre dati di loro. Restiuisce un valore in base alla valutazione di un' espressione boolenana
Sintassi: Condizione? valore1 : Valore2
verifico se la condizione è vera (true| false) -> se è vera restituisce valore1 altrimenti valore2
-negazione -> x= 5 -> -x=-5
se ci sono due valore - è sottrazione, se c'è solo un valore è negazione

-> incremento e degremento le posso apllicare solo alle varibili
-> è molto importante la posizione dell'elemento (se a sinistra oppure e destra dell' operando)
-> se l'operazione è a sinistra dell' operando viene eseguita prima di valutare l'espressione
-> se l'operazione è a destra dell' operando l'operazione viene eseguita dopo
