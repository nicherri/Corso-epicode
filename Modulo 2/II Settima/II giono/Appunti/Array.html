Array

L'array è una variabile multidimensionale. Dentro a una variabile posso inserire un valore, dentro un array poso inserire più valori

Dentro a un array possiamo incontare Strutture di dati omogenee

Dentro possiamo conservare qualsia tipo di dato ( primitivi o oggetti)

Gli array sono elenchi, liste in cuo salvare diversi dati

Dentro a un array associamo a un unic onome di variabile più valori

L'uso degli array ci semplifica le operazione nel caso in cui dobbimao definie più valiabili

Proprietà degli Array

in js esistono una serire di metodi che ci permettono di fare delle operazioni sugli array, la sintassi per fare questa cosa è nome_ array.nome_metodo

Primo Metodo è length -> conta quanti elemetni ci sono negli array

Inserire elementi in un array

per inserire degli elemnti in un array abbiamo un medoto che si chiama push

Rimuovere elementi in un array

per rimuovere degli elemnti in un array abbiamo un medoto che si chiama pop

Aggiungere e rimuovere elementi all'inizio array

per aggiungere degli elementi all'inizio array abbiamo un medoto che si chiama unshift e rimuovere dall'inizio dell'array shift shift e unshift sovrascrivono l'array -> gli indici vengono modificati
